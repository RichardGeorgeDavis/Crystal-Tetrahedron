# Crystal Tetrahedron

Standalone WebGL2 port of the Shadertoy effect [Crystal Tetrahedron](https://www.shadertoy.com/view/WsBfWt).

## Run

Open `index.html` in a WebGL2-capable browser, or serve the folder with any static file server.

## Controls

Hold and drag anywhere on the canvas to orbit the camera.
Press the spacebar to pause or resume the animation.
Use the admin panel to adjust dispersion, bounce count, focus, exposure, and playback speed.
